# Flux GitHub Action

This action sets up a [Fluxpipe](https://github.com/metrico/fluXpipe) instance to execute [Flux](https://github.com/influxdata/flux) scripts.

# Usage

```yaml
steps:
  - uses: lmangani/flux-github-action@v1.0.0
```
